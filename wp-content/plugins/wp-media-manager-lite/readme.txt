== Installation ==
1. Unzip <plugin>.zip file.
2. Upload all the files to the /wp-content/plugins/wp-media-manager.
3. Activate the plugin through the ‘Plugins’ menu in WordPress.
4. Firstly, setup default plugin’s settings, click on WP Media Manager Lite option menu in WordPress left admin menu
5. To display the folder enhancement with drag and drop method, enable WP Media Manager Lite from general settings.
